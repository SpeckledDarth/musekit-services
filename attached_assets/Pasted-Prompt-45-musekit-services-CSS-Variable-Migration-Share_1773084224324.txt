2. Fix Loading Skeletons
LoadingSkeleton component: Replace:

bg-gray-200 → bg-muted
bg-gray-100 → bg-muted
animate-pulse → keep as-is OR add CSS variable integration: animation: var(--skeleton-animation, pulse)
The --skeleton-animation CSS variable has values pulse, shimmer, or static. To integrate:

If pulse: use standard animate-pulse
If shimmer: use a shimmer gradient animation
If static: no animation
The simplest approach is to just ensure skeleton elements use bg-muted instead of bg-gray-*, and leave the animation handling to a CSS class that reads --skeleton-animation. If implementing the full animation switch is complex, at minimum replace the hardcoded grays.
