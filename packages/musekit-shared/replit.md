# @musekit/shared

This module contains shared types, utility functions, and configuration settings that are used across the MuseKit platform. It serves as the single source of truth for the application's domain models and basic utilities to ensure consistency between the frontend, backend, and other packages.

## Features
- **TypeScript Interfaces**: Definitions for users, organizations, subscriptions, etc.
- **Utility Functions**: Helpers for formatting dates, currency, class names, etc.
- **Configuration**: The application configuration file `APP_CONFIG`.

## Commands
- `npm run build` - Builds the TypeScript source into JavaScript and type definitions.
- `npm run dev` - Runs the TypeScript compiler in watch mode.
